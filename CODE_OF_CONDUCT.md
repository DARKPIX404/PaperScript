# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1.

Be respectful, constructive and inclusive. Harassment, discrimination and personal attacks are not tolerated. Report unacceptable behavior by opening a confidential issue to the maintainers.

Project maintainers are responsible for clarifying and enforcing these standards and may take any action they deem appropriate.
